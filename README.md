## Dependencies

- python-version >=3.10

## How to set up

* windows

```
$ git clone https://github.com/navidsoleymani/mock_server.git 
$ cd <project_path>
$ python -m venv venv
$ venv\Scripts\activate
(venv)$ python.exe -m pip install --upgrade pip
(venv)$ pip install -r requirements.txt
(venv)$ python run.py
```
* linux

```
$ git clone https://github.com/navidsoleymani/mock_server.git 
$ cd <project_path>
$ python -m venv venv
$ source venv/bin/activate
(venv)$ python.exe -m pip install --upgrade pip
(venv)$ pip install -r requirements.txt
(venv)$ python run.py
```
